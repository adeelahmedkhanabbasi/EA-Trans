🏥 Overview
EA-Trans is a novel transformer-based architecture designed for accurate glioma grading from 3D multimodal MRI scans. The model addresses key limitations in existing methods by integrating four specialized modules:

🔧 Key Components
EST (Edge Sensitive Tokenization): Enhances boundary representation using edge enhancement filters

SASFA (Shared Axis Spatial Features Alignment): Reduces spatial redundancy through axis-wise processing

VMWTC (Volumetric Multi-Scale Wavelet Transform Convolution): Captures shape-sensitive features across multiple scales

ASA (Adaptive Self-Attention): Dynamically prioritizes clinically relevant tokens using IQR-based selection

📁 Datasets
The model was evaluated on:

BraTS2020: Multi-institutional brain tumor segmentation dataset

UCSF-PDGM: University of California San Francisco Preoperative Diffuse Glioma MRI dataset

ADNI: Alzheimer's Disease Neuroimaging Initiative (for extended validation)
